How do I create my personal web page using Github?

 

1.       Preparation.

1)       Your personal e-mail address.

2)       Download Git bash. Please click on git-scm.com/downloads

3)       Sign up github account using your personal e-mail account.

Please click on GitHub: Lets build from here GitHub

4)       Download any kind of editor program for web-page design.

I strongly recommend Visual Basic Code.

Please click on Download Visual Studio Code - Mac, Linux, Windows

5)       Working directory for web page development on your computer.

 

2.       Design.

1)       Create index.html and save your working directory.

When someone visits your website, the index.html page invokes the first time.

Therefore, index.html is like a gate in the house.

 

 

Example.

 

<html>

<h1> Your names portpolio </h1>

<Img src=./sample.jpg>

</html>

 

 

2)       Preparing your picture. Define picture file name is yourname.jpg.

 

3.       Set up a repository on the github.

1)       Sign in on github which enter your github account.

2)       Click on new repository.

3)       Enter your name which is yourname.github.io for repository address on the repository name.

The yourname.github.io will be used web page address.

 

4)       Select Public option for next selection.

5)       Click on Create repository button.

6)       Click on uploading an existing button on the next page.

7)       Drag your index.html and yourname.jpg files to upload window.

8)       Click on Commit changes button.

 

4.       Web page check.

1)       Check your website opening using yourname.github.io on the browser.

2)       If you find any other bug, please review and feedback its.
